Self-Healing

Detects failures and auto-recovers. Disk full? Auto-prune. Service down? Auto-restart. You sleep, homebutler watches.

# alerts.yaml
alerts:
  - name: disk-cleanup
    condition: disk_usage > 90%
    action: docker system prune -f
  - name: restart-nginx
    condition: !http_ok("localhost:80")
    action: systemctl restart nginx
